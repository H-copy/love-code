vDirective

  • vDirective(type: VPropType, directive: VDirective, value?: any): VDirectiveProp
    • Defined in props/pro.ts:75

    创建基础指令对象

    summary

    这里将指令看做一类 vue 标签属性, 通过指令配置生成对应的 属性名称, 例如： { name: 'v-on', arg: 'click', modifiers: ['stop] } => 'v-on:click.stop'

    example
    vDirective(VPropType.DIRECTIVE, { name: 'h-size', arg: 'full' }, { width: 100, height: 100 })
=> {
   name: 'h-size:full',
   value: { width: 100, height: 100 },
   type: '__DIRECTIVE',
   isProp: '__Prop',
   directive: { name: 'h-size', arg: 'full' } // 指令配置缓存, 提供给渲染器生成对应指令
 }

    Parameters

    • type: VPropType

      属性类型

    • directive: VDirective

      指令配置

    • Optional value: any

      属性值

    Returns VDirectiveProp
